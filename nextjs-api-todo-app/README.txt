# nextjs-api-todo-app

Next.jsで「TODOリスト」を作成してください。
-「簡易アプリ仕様設計-SampleTODOリスト」のようにページを分けて作ってみましょう。
- todo一覧
- todo詳細
- todo作成
- todo編集画面
- Next.jsのAPI Routesを使い、Restを使ったCRUD機能の実装
- MaterialUI
- NextAuthを使った認証機能
  - ログイン
  - ログアウト
  - サインアップ
- useContextもしくはRecoilを用いた状態管理
- Typescriptを導入
- DBの導入(postgres)

余裕があれば
- TypeScriptのanyを使わずに実装
- UIフレームワークを使わずに実装
